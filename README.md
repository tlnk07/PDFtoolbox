# 🛠️ PDFtoolbox

Take full control of your PDF workflow. Easily combine multiple files or restructure pages exactly how you need them.

## ✨ Features


### Real-Time Transcription
- **Merge PDFs** – Combine two or more PDF files into one document.
  - Drag-and-drop reordering of files.
  - Display file names and sizes.
  - Real-time progress bar and status updates.
  - Option to open the output folder after successful merge.
- **Split & Reorder PDF** – Extract specific page ranges from a PDF and rearrange them into a new PDF.
  - Load any PDF and see total page count.
  - Define multiple page ranges (parts) with start/end pages.
  - Reorder parts using intuitive up/down buttons.
  - Progress bar and status during processing.
  - Save the reordered PDF as a single file.
- **Modern GUI** – Dark theme, responsive layout, and clean design using CustomTkinter.
- **Multi-threaded** – PDF processing runs in the background, keeping the UI responsive.
- **Cross-platform** – Works on Windows, macOS, and Linux (where Python and dependencies are installed).


## 📦 Requirements
- Python 3.7 or higher
- CustomTkinter
- PyPDF2
- pypdf


## 🔧 Installation
- Clone or download this repository.
- Install the required packages using pip:
```bash
pip install -r requirements.txt
```
- Run the application:
```bash
python main.py
```
